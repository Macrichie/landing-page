# Landing Page (HTML + CSS)

This project is a clean, responsive landing page built with plain **HTML + CSS** and inspired by the Figma community file:

- `https://www.figma.com/design/3DkTlqjzP4X3YDRm5xk0kV/Landing-Page-UI-Kit---Fully-customizable-landing-page-UI-kit---Export-as-HTML--Community-?node-id=0-1&p=f&t=yf84Xp53xWrPVbJe-0`

## Files

- `index.html` — page structure + a tiny script for the mobile menu
- `styles.css` — all styling (theme variables, layout, responsive rules)

## How to view

### Option A: Open directly

Double click `index.html` to open it in your browser.

### Option B: Run a local server (recommended)

If you have Python:

```bash
python -m http.server 5173
```

Then open `http://localhost:5173`.
